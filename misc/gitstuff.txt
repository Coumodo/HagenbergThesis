Clone from origin:
	git clone https://github.com/Digital-Media/HagenbergThesis.git

List current remotes:
	git remote -v

Add a new remote:
	git remote add <name> <url>
	git remote add overleaf https://www.overleaf.com/5430778jcfjpz

Push to remote:
	git push origin
	git push overleaf
	git push --all
